# Pack 182 Branding Assets

Place your Pack 182 logo here.

## Requirements:
- **Format:** PNG (with transparency) or SVG
- **Size:** 500x500 pixels minimum
- **Max file size:** < 200KB
- **Quality:** High resolution, crisp edges

## File Names:
- `pack-logo.png` (primary logo)
- `pack-logo.svg` (vector version, optional)
- `pack-logo-white.png` (white version for dark backgrounds, optional)

## Tips:
- PNG with transparent background works best
- SVG is ideal if you have it
- Make sure logo is high quality (not pixelated)

See `/ASSET_PLACEMENT_GUIDE.md` for complete instructions.
