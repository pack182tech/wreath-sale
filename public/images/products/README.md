# Wreath Product Photos

Place your wreath product photos here.

## Requirements:
- **Format:** JPG or PNG
- **Size:** 800x800 pixels (square) recommended
- **Max file size:** < 500KB per image
- **Quality:** Clear, well-lit photos

## Naming Convention:
Use descriptive lowercase names with hyphens:
- `red-ribbon-wreath.jpg`
- `green-ribbon-wreath.jpg`
- `gold-ribbon-wreath.jpg`
- etc.

## Example Files:
```
red-ribbon-wreath.jpg
green-ribbon-wreath.jpg
burgundy-ribbon-wreath.jpg
navy-ribbon-wreath.jpg
```

See `/ASSET_PLACEMENT_GUIDE.md` for complete instructions.
